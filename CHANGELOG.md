# Changelog

All notable changes to Aurora Music will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project structure
- MPV-based audio playback engine
- SQLite database with SQLAlchemy ORM
- Library scanner with Mutagen metadata extraction
- Automatic folder monitoring with Watchdog
- PySide6+QML modern UI with glassmorphism design
- Sidebar navigation
- Playlist management
- 10-band equalizer with presets
- DSP features (ReplayGain, crossfade, normalization)
- Theme system with multiple themes
- AI-powered recommendations using librosa
- Plugin system with base classes
- Test suite with pytest
- Packaging configurations for DEB, RPM, Flatpak, AppImage
- Branding assets (logo, splash screen)

## [0.1.0] - 2024-01-01

### Added
- Initial release
- Core playback functionality
- Library management
- Basic UI
- Equalizer
- Theme system
- Plugin system
- Packaging
