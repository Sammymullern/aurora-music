# Icon Assets

This directory contains icon assets for Aurora Music.

## Icon Categories

- **app-icons**: Application icons for different platforms (Linux, Windows, macOS)
- **ui-icons**: UI icons for the interface (play, pause, skip, etc.)
- **file-icons**: File type icons (MP3, FLAC, etc.)

## Icon Guidelines

- Use SVG format for scalability
- Follow the Aurora color scheme:
  - Primary: #7c3aed (purple)
  - Secondary: #3b82f6 (blue)
  - Accent: #ec4899 (pink)
- Maintain consistent stroke width and corner radius
- Use 24x24 base size for UI icons
- Use 64x64 or larger for application icons

## Icon Naming Convention

- UI icons: `icon-{name}.svg`
- File icons: `file-{format}.svg`
- App icons: `app-{size}x{size}.png`
